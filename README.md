# Simple React Blog UI

A simple blog interface built with React to practice component-based architecture, API data fetching, and clean UI design.

## Live Demo
https://react-blog-ui-eight.vercel.app/

## Tech Stack
- React.js
- Tailwind CSS
- JavaScript
- JSONPlaceholder API

## Features
- Blog post listing
- Single post view
- Clean and responsive layout
- Reusable components

## Purpose
This project was built to strengthen my frontend development skills and demonstrate practical use of React in building user-focused interfaces.
